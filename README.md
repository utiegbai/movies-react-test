# movies-react-test

## Requirements
NodeJS, NPM, Radis server.

## How to to run this project

Clone the repo to your machine
### `git clone https://github.com/utiegbai/movies-react-test.git`

### `cd movies-react-test`

## Run Worker
### `cd worker`
### `npm run dev`

## Run API
### `cd api`
### `npm run dev`

## Run Client
### `cd client`
### `yarn start`
